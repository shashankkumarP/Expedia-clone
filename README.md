## API Reference

#### Get all items

```http
  GET /api/items
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `datefilter2` | `name` | Range of dates |

#### Get item

```http
  GET /api/items/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `datefilter`      | `name` | it gives date range  |


#### add(num1, num2)

Takes two numbers and returns the sum.
